How to play the game ?  <br/>
R -> 甩竿    <br/> 
滑鼠右鍵點擊海中任一處 -> 選擇地方開始釣魚<br/>
連擊空白鍵 -> 加速魚上鉤<br/>
滑鼠左鍵 -> 拉魚<br/>
上下鍵 -> 加速/減速拉魚(需和左鍵同時按)<br/>
左上方按鍵(Restart) -> 重來<br/>
